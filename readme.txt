== Description ==
Easily allows your blog post or page to be retweeted. Currently being used by SocialBrite and other members of the social media community.

= Features =

* Allows you to change the source which you retweet, e.g. "<the title> <the url> via @yourname"

* Easily installation and customization

* Better integration, allowing custom titles, related accounts

* Integrates with Wordpress MU


== Installation ==

Follow the steps below to install the plugin.

1. Upload the TweetButton directory to the /wp-content/plugins/ directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to TweetButton->Settings to configure the button
